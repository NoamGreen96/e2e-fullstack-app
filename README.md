# Auth App with TiDB, Kafka and Docker

##
- Frontend: HTML + JavaScript
- Backend: Node.js + Express
- Database: TiDB (MySQL compatible)
- Message Queue: Kafka (via Docker)
- Logging: log4js
- Containerization: Docker & Docker Compose

## How to check logs 
docker logs /backend
